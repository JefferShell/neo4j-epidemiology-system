## 项目简介

本系统是一套关系图谱展示和编辑平台，可用于可视化和管理各种关系数据。系统提供用户登录、图谱展示、节点编辑和删除等功能。

## 主要功能

- 关系图谱展示
- 图谱节点编辑与删除
- 图片查询和展示
- 基于三元组数据的图谱查询
- 班组节点的属性修改和删除
- 班组节点的鼠标悬浮信息展示
- 班组节点的增加


## 技术架构

系统使用以下技术和工具：

- 后端：Python，MySQL
- 前端：JavaScript，React


## 使用说明

1. 下载并解压项目包。
2. 启动项目。
3. 在浏览器中访问项目地址。
4. 登录注册系统，并修改密码。
5. 在首页，使用三元组数据查询图谱。
6. 点击班组类型的节点，进入编辑页面，对班组的属性进行修改。
7. 可以对班组进行删除。
